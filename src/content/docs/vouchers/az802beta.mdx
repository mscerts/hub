:::note
The first 300 people who take Exam AZ-802 (beta) on or before July 22, 2026, can get 80% off market price using code **AZ802Chandler**.
:::

Exam AZ-802 is the new beta exam for Microsoft Certified: Windows Server Administrator Associate. It replaces both Exam AZ-800 and Exam AZ-801, consolidating them into a single exam.

## How to claim the 80% discount

<Steps>

1. Register for [Exam AZ-802 (beta)](https://learn.microsoft.com/credentials/certifications/exams/az-802?WT.mc_id=studentamb_165290).
2. Schedule to take the exam on or before **July 22, 2026**.
3. When prompted for payment, use the code **AZ802Chandler** to receive the 80% discount.

</Steps>

:::tip
This is not a private access code. The seats are offered on a first-come, first-served basis to the first 300 people. This discount is not available in Turkey, Pakistan, India, or China.
:::
